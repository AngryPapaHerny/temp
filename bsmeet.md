# 회의 내용

### 14:00 회의 시작

김현우:
오늘 회의의 목적은 **AI를 활용한 데이터 분석 교육 프로그램을 설계하는 것**입니다. 특히 고등학생 대상 프로그램을 중심으로 커리큘럼을 구성하는 것이 목표입니다.

이지은:
최근 학생들이 **ChatGPT 같은 생성형 AI 도구를 많이 사용하고 있어서**, 단순 코딩 교육보다는 **AI 도구를 활용한 문제 해결 중심 교육**이 필요할 것 같습니다.

박민수:
동의합니다. 실제 데이터 분석 교육에서는

* 데이터 수집
* 데이터 정제
* 데이터 분석
* 데이터 시각화

이 네 단계의 워크플로우를 이해시키는 것이 중요합니다.

---

### 14:15 교육 대상 논의

최서연:
교육 대상은 고등학생으로 설정했는데, **프로그래밍 경험이 없는 학생들도 참여할 수 있도록 해야 합니다.**

정다훈:
그렇다면 Python 문법을 깊게 가르치기보다는

* Pandas 기본 사용법
* 간단한 데이터 시각화
* AI를 활용한 데이터 해석

이 정도 수준이 적절할 것 같습니다.

김현우:
좋은 의견입니다. 특히 **AI에게 데이터 분석을 질문하는 방법(프롬프트 작성)**도 교육에 포함하면 좋겠습니다.

---

### 14:35 실습 데이터 논의

박민수:
실습 데이터는 학생들이 흥미를 느낄 수 있도록 **타이타닉 데이터셋이나 넷플릭스 콘텐츠 데이터** 같은 것이 좋을 것 같습니다.

이지은:
타이타닉 데이터는 교육에서 많이 사용되지만, 이미 많이 알려져 있어서 **학생들이 흥미를 느끼지 못할 수도 있습니다.**

최서연:
그렇다면 다음과 같은 데이터도 고려할 수 있습니다.

* K-pop 음원 데이터
* 유튜브 조회수 데이터
* 게임 사용자 데이터

정다훈:
학생들이 좋아할 만한 데이터라면 **게임 데이터 분석 프로젝트**도 좋을 것 같습니다.

---

### 15:00 AI 활용 방법 논의

이지은:
AI를 단순한 답변 도구가 아니라 **데이터 분석 보조 도구**로 활용하도록 가르치는 것이 중요합니다.

예를 들어 학생들이 다음과 같은 질문을 할 수 있습니다.

* 이 데이터에서 가장 중요한 변수는 무엇인가?
* 생존율에 가장 영향을 준 요소는 무엇인가?
* 데이터에서 발견되는 패턴은 무엇인가?

박민수:
맞습니다. 특히 **EDA(Exploratory Data Analysis)** 과정에서 AI를 활용하면 좋을 것 같습니다.
