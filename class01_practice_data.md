# 1차시 실습용 샘플 데이터

## 실습 A: 회의록 작성 실습용 회의 녹취록

> **사용 시점**: 1차시 3교시 — 이메일/회의록 작성 실습
> **실습 방법**: 아래 녹취록을 ChatGPT 또는 Claude에 붙여넣고 회의록 작성을 요청

---

### 녹취록 원문

```
[2025년 3월 5일 수요일 오후 2시, 마케팅팀 주간 회의]
[참석자: 박지수 팀장, 이민호 과장, 김수진 대리, 정우성 사원]

박지수: 자, 다들 오셨으면 시작할게요. 오늘 안건은 세 가지예요.
        2월 성과 공유, 3월 캠페인 계획, 그리고 새 협업 툴 도입 건이에요.

        이민호 과장님, 2월 실적부터 공유해주시겠어요?

이민호: 네. 2월 총 매출은 4억 2천만 원으로, 목표 대비 103% 달성했습니다.
        특히 SNS 광고 채널이 좋았어요. 인스타그램에서 클릭률이 전월 대비 18% 올랐고,
        유입 고객 중 신규 비율이 62%였습니다.
        아쉬운 점은 전환율인데요, 유입은 많았는데 실제 구매로 이어진 비율이 2.3%로
        전월 2.8%보다 떨어졌어요.

박지수: 전환율 하락 원인 파악됐나요?

이민호: 랜딩 페이지 로딩 속도 이슈가 있었던 것 같아요. 개발팀에서 확인 중입니다.
        3월 5일까지 수정 완료 예정이라고 했어요.

박지수: 알겠어요. 수정 완료되면 저한테 공유해주세요.
        다음, 김수진 대리님, 3월 캠페인 계획 발표해주시겠어요?

김수진: 네. 3월 캠페인은 화이트데이 시즌을 활용한 기획이에요.
        3월 7일부터 14일까지 총 8일간 "선물의 이유" 테마로 진행합니다.
        예산은 2,500만 원 배정했고요.

        실행 채널은 인스타그램, 유튜브 쇼츠, 네이버 블로그 세 곳이에요.
        인플루언서 협업도 계획 중인데, 10만~50만 팔로워 미드 티어 5명 섭외
        진행 중입니다. 현재 3명 컨펌받은 상태고요.

박지수: 목표 KPI는요?

김수진: 캠페인 기간 매출 목표 8천만 원, 인스타그램 팔로워 500명 증가,
        해시태그 누적 게시물 1,000개 이상입니다.

박지수: 좋아요. 인플루언서 계약서는 법무팀 검토 받는 거 잊지 마세요.

김수진: 네, 이미 요청해뒀어요. 이번 주 내로 회신 올 것 같아요.

박지수: 마지막으로 협업 툴 도입 건. 정우성 사원, 비교 결과 공유해줘요.

정우성: 네. 슬랙, 노션, 팀즈 세 가지를 비교했는데요.
        비용 면에서는 팀즈가 기존 MS365 라이선스에 포함돼서 추가 비용이 없어요.
        노션은 멤버당 월 8달러, 슬랙은 멤버당 월 7.5달러입니다.
        기능 면에서는 노션이 문서 관리와 협업이 가장 강점이고,
        슬랙은 커뮤니케이션 위주라 문서 관리가 약해요.
        팀즈는 화상 회의랑 MS 오피스 연동이 강점이에요.

박지수: 우리 팀 상황에서는 문서 협업이 제일 필요하니까 노션이 맞겠네요.
        비용은 한 달 파일럿 해보고 결정하죠.
        정우성 사원이 노션 파일럿 계정 3개 만들어서 이번 주 안에 팀원들 초대해줘요.

정우성: 알겠습니다.

박지수: 오늘 결정 사항 정리하면,
        하나, 랜딩 페이지 수정 완료 후 이민호 과장이 결과 공유.
        둘, 인플루언서 계약서 법무팀 검토.
        셋, 노션 파일럿 정우성 사원이 진행.

        다음 회의는 3월 12일 같은 시간이에요. 수고하셨습니다.
```

---

### 실습 프롬프트 예시

```
위 회의 녹취록을 다음 형식의 공식 회의록으로 정리해줘:

1. 회의 기본 정보 (일시, 장소, 참석자, 주제)
2. 안건별 논의 내용 요약
3. 결정 사항
4. 액션 아이템 표 (담당자 / 할 일 / 마감일 / 상태)
5. 다음 회의 예정 일정
```

## 실습 B: 이메일 작성 실습용 상황 카드

> **사용 시점**: 1차시 3교시 — 비즈니스 이메일 작성 실습
> **실습 방법**: 아래 상황 중 하나를 골라 AI에 이메일 작성 요청

---

### 상황 1: 납기 연장 요청

```
상황 설명:
- 나의 역할: 소프트웨어 개발사 PM (프로젝트 매니저)
- 고객사: 한빛물류 / 담당자: 이현수 부장
- 계약 내용: 물류 관리 시스템 커스터마이징
- 원래 납기: 2025년 3월 15일
- 변경 요청 납기: 2025년 3월 28일 (2주 연장)
- 연장 이유: 고객사가 요청한 추가 기능 2개 (실시간 위치 추적, 모바일 앱 연동)
             개발 범위가 당초보다 40% 늘어남
- 추가 비용: 없음 (당사 부담)
- 중간 산출물 공유: 3월 20일 베타 버전 먼저 전달 예정
```

### 상황 2: 계약 갱신 제안

```
상황 설명:
- 나의 역할: 클라우드 솔루션 영업 담당
- 고객사: ABC 제조 / 담당자: 최영훈 IT팀장
- 현재 계약: 서버 호스팅 서비스, 3월 31일 만료
- 갱신 제안 내용:
  * 기존 연 480만 원 → 갱신 시 연 450만 원 (6% 할인)
  * 저장 용량 1TB → 2TB 무상 업그레이드
  * 24시간 기술 지원 추가 (기존엔 평일만)
- 목표: 3월 10일까지 갱신 확정
```

### 상황 3: 미팅 일정 조율

```
상황 설명:
- 나의 역할: 마케팅 에이전시 AE (Account Executive)
- 상대방: 신규 잠재 고객사 스타트업 / 담당자: 박서연 대표
- 목적: 브랜딩 제안서 발표 미팅 (30분 소요)
- 가능 일정: 3월 10일(월), 11일(화), 13일(목) 오전 10시~12시
- 미팅 형식: Zoom 온라인 또는 강남 사무실 (수강생 선택)
- 참고: 박 대표는 바쁜 스타트업 대표, 간결하고 존중하는 톤 필요
```

## 실습 C: AI 모델 비교 테스트 질문 세트

> **사용 시점**: 1차시 2교시 — ChatGPT, Claude, Gemini 비교 실습
> **실습 방법**: 아래 질문들을 3개 AI 툴에 동일하게 입력하고 결과 비교

---

### 비교 질문 세트

**[창의성 테스트]**
```
우리 회사가 B2B SaaS 기업인데, 고객 이탈률을 줄이기 위한
비전통적인 아이디어 5가지를 제안해줘.
```

**[논리/추론 테스트]**
```
A팀은 B팀보다 생산성이 20% 높다.
C팀은 A팀보다 생산성이 15% 낮다.
D팀은 C팀보다 생산성이 30% 높다.
네 팀을 생산성 순서대로 나열해줘.
```

**[한국어 이해 테스트]**
```
다음 문장에서 어색한 부분을 찾아 자연스럽게 고쳐줘:
"저희 회사는 고객들한테 최선을 다하여 서비스하고 있으며,
앞으로도 더욱 더 발전하는 모습 보여드리도록 하겠습니다."
```

**[최신 정보 한계 테스트]**
```
오늘 날짜 기준 코스피 지수를 알려줘.
```

**[환각(Hallucination) 테스트]**
```
2024년에 한국에서 개봉한 영화 "서울의 봄 2"의 줄거리를 설명해줘.
```

## 실습 D: 브레인스토밍 실습 시나리오

> **사용 시점**: 1차시 3교시 — AI 브레인스토밍 실습
> **실습 방법**: 아래 시나리오 중 하나 선택하여 AI에 브레인스토밍 요청

---

```
시나리오 1 (스타트업):
우리 팀은 30대 직장인을 타겟으로 한 독서 습관 앱을 출시 예정.
MAU(월간 활성 사용자) 1만 명 달성을 위한 초기 마케팅 전략 아이디어를 내줘.

시나리오 2 (중소기업):
오프라인 반찬 가게를 운영 중인데, 배달 서비스로 매출을 늘리고 싶어.
경쟁 많은 배달 시장에서 차별화할 수 있는 아이디어 10가지를 제안해줘.

시나리오 3 (대기업):
HR팀인데 MZ세대 직원들의 이직률이 높아지고 있어.
비금전적 방법으로 직원 만족도와 유지율을 높이는 아이디어를 제안해줘.
```
